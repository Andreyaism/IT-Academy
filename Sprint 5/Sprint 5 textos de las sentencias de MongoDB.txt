2. Mostra'm tots els documents de les pel·lícules produïdes en 1932, però que el gènere sigui drama o estiguin en francès.

db.movies.find( { $and: [
                         { $or: [ {genres: 'Drama'}, {languages: 'French'} ] },
                         { year: 1932 } ]
                }, {year: 1, genres: 1, languages: 1, _id: 0} )


3.  Mostra'm tots els documents de pel·lícules estatunidenques que tinguin entre 5 i 9 premis que van ser produïdes entre 2012 i 2014.

db.movies.find( { countries: 'USA', 
                  'awards.wins': { $gte: 5, $lte: 9}, 
                  year: { $gte: 2012, $lte: 2014 }  }, { countries: 1, 'awards.wins': 1, year: 1})

1. Compte quants comentaris escriu un usuari/ària que utilitza "GAMEOFTHRON.ES" com a domini de correu electrònic.

db.users.countDocuments( {email: {$regex: /@gameofthron.es/} })

2. Quants cinemes hi ha en cada codi postal situats dins de l'estat Washington D. C. (DC)?

db.theaters.aggregate( [
   { $match: { "location.address.state": "DC" } }, 
   { $group: { _id: "$location.address.zipcode", num_cinemas:{$sum: 1} } } ] )
